jQuery

Insert the following code directly below the included CSS and Javascript files and before closing the </head> in your HTML document.


<script type="text/javascript">
    $(document).ready(function(){
        $('#demo').Griddler({
		columns: 3,
		featureGrid: 'disable',
		gridLimit: false,
		expandLimit: false
	});
    });
</script>

For more information on customizing the Griddler plugin options, read the Javascript documentation section.

HTML

The Griddler content in this demo fills the entire width of the browser window. This is done by setting <div id="demo_parent"> as a direct child of the <body>. <div id="demo_parent"> is assigned a width of 100%, which is a percentage of its parent container, the <body> element.

The following code enables the lightbox feature to disply video content from Vimeo by giving the <a> tags a class of "iframe". It is important to note that while the lightbox is enabled, it is done with the "iframe" class and not the "lightbox" class seen in other demos. As of Griddler version 1.0.1, Youtube and Vimeo are supported video services. Support of other services will likely be included in future updates. Youtube and Vimeo implementation is covered in more detail below.

                       
<body> 

<!---Other Outside HTML Content--->
                
<div id="demo_parent" style="width:100%;">         
    <ul id="demo">
        <li>
          <a href="http://player.vimeo.com/video/37066208" class="lightbox">
            <img src="http://b.vimeocdn.com/ts/254/463/254463200_640.jpg" alt="" />
           </a>
           <div class="project_info">
             <h2>Didot by Roland Budai</h2>
           </div>
        </li>
        
        <li>
          <a href="http://player.vimeo.com/video/37045082" class="lightbox">

To embed a Vimeo video, use the link found in Vimeo's embed feature. As of this writing, Vimeo provides an <iframe> markup. Copy Vimeo's supplied <iframe> source value and apply it to your <a> tag's href value. Note that the image thumbnails in the following two examples are not working links. Your video will have a different link for the video's thumbnail image.

        		              
<a href="http://player.vimeo.com/video/37066208" class="iframe">
   <img src="image/path_to/thumbnail.jpg" alt="" />
</a>

Embedding a Youtube video is identical to the steps for embedding a Vimeo video. Below is an example of displaying your Youtube videos in the Griddler plugin.

        		              
<a href="http://www.youtube.com/embed/XmIIgE7eSak" class="iframe">
   <img src="image/path_to/thumbnail.jpg" alt="" />
</a>
