jQuery

Insert the following code directly below the included CSS and Javascript files and before closing the </head> in your HTML document.


<script type="text/javascript">
    $(document).ready(function(){
        $('#demo').Griddler({
		columns: 3,
		featureGrid: 'top',
		rollover: false,
		expandLimit: false
	});
    });
</script>

In this example, a three column layout is described, with a large top featured grid with no hover animation since the value assigned to rollover is false. For more information on customizing the Griddler plugin options, read the Javascript documentation section.
