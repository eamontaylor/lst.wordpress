Quick Start

To start your installation of the Griddler jQuery Plugin, copy all of the necessary files from the 'plugin_files' folder to your desired web document folder. The quick start guide assumes that these necessary files are in the same folder directory, as shown in the 'plugin_files' folder. If you have moved the javascript, CSS or image files into a different folder, then you will have to modify the links to those respective files.

HTML head

Start with a new HTML document. Inside the <head> of your HTML document, you will need to include the necessary CSS and Javascript files. Start by including the Griddler CSS file.

<link rel="stylesheet" type="text/css" href="griddler_style.css" />

The plugin requires the jQuery javascript library. So include the 'jquery.1.8.0.min.js' javascript file directly below the CSS file.

<script type="text/javascript" src="jquery.1.8.0.min.js"></script>

On the next line, include the minified Griddler plugin.

<script type="text/javascript" src="griddler.1.1.0.min.js"></script>

Finally, insert the following jQuery into the head of your document (before the closing </head> tag).


<script type="text/javascript">
    $(document).ready(function(){
        $('#demo').Griddler();
    });
</script>

This will conclude the code needed inside the <head> of your HTML document. For more information on customizing the Griddler plugin options, read the Javascript documentation section.

That's all you need to get your Griddler plugin installed! Save your HTML document and view it in your web browser.

The plugin will automatically replace <ul id="demo"> with the Griddler content and will fill the width of the parent container, that is the HTML element that is directly containing <ul id="demo">.

It is also important to note that each <li> anchor tag has a class attribute of "lightbox". This is all you need to enable the lightbox for that particular grid (<li>). This is done on each individual <li> anchor tag. This gives you the flexibility to control which grid will have a lightbox when clicked and which grids can be sent to a direct URL.
