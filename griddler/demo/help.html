Help/FAQ

This page is here to help you troubleshoot any problems you may encounter after purchasing the Griddler jQuery Plugin. Make sure that you have carefully read the Documentation section. If you're still having issues after reading this help file and the documentation, please feel free to send us an email using the contact form on our Codecanyon profile page, or leave a comment in this item's discussion section.

Griddler is not rendering

If your Griddler plugin is not rendering inside of the browser, check the following:

  1. Is the jQuery library included in the page and positioned BEFORE the Griddler plugin?
  2. Is the Griddler plugin file included in the page?
  3. Is the <ul> tag with the correct ID included in the page between the "body" tags?
  4. Make sure all of CSS and Javascript files in the <head> have the correct link in their src or href attributes.

The lightbox isn't working

If your Griddler lightbox isn't showing, check the following:

  1. Does the particular list item's <a> tag have a class of "lightbox"?
  2. Does the list item's <a> tag have the correct href value?
  3. If the screen overlay isn't visible, go to line 101 of your 'griddler_style.css' document. Make sure the URL path is correct.
  4. In some cases, Internet Explorer 7 and 8 may experience unreliable execution of jQuery's .load() event handler function, causing the lightbox to stall or the image will fail to appear in the lightbox area. This issue has been documented in jQuery's library. More information can be found here.

My video isn't showing in the lightbox

If your video isn't display in the lightbox, check the following:

  1. Make sure that the desired list item's <a> tag has a class of "iframe".
  2. Make sure the <a> tag has the correct URL in the href attribute
  3. Version 1.0 only supports Vimeo and Youtube embeds at this time.
  4. Make sure your <a> tag has the correct markup as described in Demo 3.
  5. Some video embeds may take a moment to display, please be patient.

The hover/rollover animation isn't working

If your Griddler rollover animation is not showing when you hover over a grid, check the following:

  1. Make sure that <div class="project_info"> is included inside the list item and the <h2> has text inside of it.
  2. Check your jQuery to make sure that the plugin's "rollover" value is set to true.
  3. If the text is visible but the overlay isn't, go to line 67 of your 'griddler_style.css' document. Make sure the URL path is correct.

Some of my grids are missing

If some of your grids are not showing in the browser, check the following:

  1. In your jQuery, check the plugin's "gridLimit" and "expandGrid" values.
  2. If "gridLimit" is not false, or a numeric value of 0, then make sure that "expandGrid" is set to true
  3. If "gridLimit" is false or 0, give it a numeric value greater than 0. Set "expandGrid" to true to ensure more grids can be shown if your grid quantity exceeds the "gridLimit" value. This is explained in Demo 1.

My Griddler content isn't the right size

If Griddler content is too small/big:

  1. The Griddler content automatically fills the full width of it's parent element.
  2. The Griddler does not have a set height, but it can be controlled by setting a "gridLimit" value in your jQuery plugin.

My grids are not the right size

If your individual grids aren't the correct size, check the following:

  1. Make sure that the "ratio" setting in your jQuery is a string datatype (surrounded by quotation marks).
  2. The "ratio" setting must have two numbers, separated by a colon (":"). Example: '12:5'
