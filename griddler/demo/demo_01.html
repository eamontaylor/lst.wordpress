jQuery

Insert the following code directly below the included CSS and Javascript files and before closing the </head> in your HTML document.


<script type="text/javascript">
    $(document).ready(function(){
        $('#demo').Griddler({
		ratio: '1:1',
		columns: 4,
		featureGrid: 'disable',
		gridLimit: false,
		expandLimit: false
	});
    });
</script>

In this example, the grid ratio is square (1:1), a four column grid layout, all grids are uniform in size because the featureGrid option is disabled. Since gridLimit is false, all possible grids are displayed and the expandLimit option returns false because there are no more grids to load.

The ratio value requires a string datatype with a colon between the width and height ratio values.

If gridLimit is assigned a number value of 0, then it is the equivalent of providing a false boolean value - displaying all available grids. For more information on customizing the Griddler plugin options, read the Javascript documentation section.
